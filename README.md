# Minimalist Markdown Novel Reader

แอปพลิเคชันเว็บแบบ Minimalist และสะอาดตาสำหรับเปิดอ่านนิยาย **"Dorea's Princess Journey (การเดินทางของเจ้าหญิงมังกรโดเรีย)"** จากไฟล์ Markdown (`.md`) โดยตรง มีฟังก์ชันอำนวยความสะดวกในการอ่านอย่างครบครัน

## 🌟 ฟังก์ชันการใช้งาน
1. **การแสดงผลสไตล์ Minimalist**: แทบไม่มีสีอะไรเลย เน้นความสะอาดตาและการจัดหน้าสรีรวิทยาการอ่านที่เหมาะสมที่สุด (Max width 660px, Line-height 1.8)
2. **ธีมปรับเปลี่ยนได้**: สลับระหว่าง ธีมสว่าง (Light) และ ธีมมืด (Dark) ได้อย่างรวดเร็ว (บันทึกจำธีมที่เลือกอัตโนมัติ)
3. **การสลับฟอนต์**: เปลี่ยนการแสดงผลฟอนต์ระหว่าง Sans-serif (ฟอนต์สไตล์โมเดิร์นไม่มีหัว) และ Serif (ฟอนต์สไตล์ดั้งเดิมมีหัว)
4. **การปรับขนาดอักษร**: เพิ่ม (A+) และลด (A-) ขนาดของตัวอักษรให้อ่านง่ายขึ้น
5. **เมนูสารบัญพับเก็บได้ (Sidebar menu)**: คลิกเปิดดูสารบัญตอนทั้งหมดเพื่อสลับบทอ่านได้อย่างรวดเร็ว
6. **ปุ่มนำทางล่างสุด**: ปุ่มตอนก่อนหน้า (Previous) และตอนถัดไป (Next) ช่วยให้สลับตอนได้สะดวกโดยไม่ต้องเปิดเมนู
7. **แถบความคืบหน้า (Progress Bar)**: เส้นทึบสีดำ/ขาวขนาด 2px ด้านบนสุด แสดงระดับการเลื่อนอ่านหน้าเพจปัจจุบัน

## 🚀 วิธีเปิดใช้งาน

เนื่องจากความปลอดภัยของเว็บบราวเซอร์ บราวเซอร์จะบล็อกการดึงข้อมูลไฟล์ `.md` ภายนอกเมื่อเปิดไฟล์ตรง ๆ ด้วยระบบ `file://` (เช่น ดับเบิลคลิกไฟล์ `index.html`) ดังนั้นคุณจำเป็นต้องเปิดโปรแกรมผ่าน **Local Web Server** ครับ

คุณสามารถเปิดเซิร์ฟเวอร์ได้อย่างง่ายดายด้วยวิธีต่อไปนี้:

### วิธีที่ 1: รันผ่าน Python (ง่ายที่สุดหากติดตั้ง Python)
เปิด Command Prompt หรือ PowerShell ในโฟลเดอร์นี้ แล้วพิมพ์คำสั่ง:
```bash
python -m http.server 8000
```
จากนั้นเข้าเว็บบราวเซอร์แล้วไปที่ลิงก์:
👉 [http://localhost:8000](http://localhost:8000)

### วิธีที่ 2: ใช้ Extension บน VS Code
1. เปิดโฟลเดอร์นี้ใน VS Code
2. ติดตั้ง Extension ชื่อ **"Live Server"**
3. คลิกขวาที่ไฟล์ `index.html` แล้วเลือก **"Open with Live Server"**

## 📂 โครงสร้างโฟลเดอร์
* `index.html` - หน้าอ่านนิยายหลัก (HTML/CSS/JS)
* `data/` - โฟลเดอร์เก็บข้อมูล
  * `chapters.json` - ดัชนีสารบัญระบุรายการตอนทั้งหมด
  * `Chapter_XX_... .md` - ไฟล์เนื้อหาบทประพันธ์แบบ Markdown
